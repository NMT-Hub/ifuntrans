# 机器翻译引擎对接文档
[[_toc_]]

## 接口文档
具体请求示例请参考：[接口文档](http://10.1.75.111:8888)

- `/translate`: 翻译接口
    - 请求参数说明
        - `engine`: 引擎名称
        - `source`: 原语言编码，可以指定`auto`为自动识别
        - `target`: 目标语言编码
        - `text`: 待翻译的文本
    - 返回参数说明
        - `source`: 原语言编码，一般与请求参数中一致。如果请求参数中为"auto"，这返回检测到的原文语种编码
        - `target`：目标语种编码，与请求参数中一致
        - `translation`: 译文
        - `engine`: 一般与请求参数中一致，如果指定引擎不支持指定的语向，则会自动选择一个支持的引擎

- `/translate/support_languages`: 获取某个引擎支持的语种编码
    - 请求参数说明
        - `engine`: 引擎名称
    - 返回参数说明
        - List[str]: 支持语种的英文名称
- `/translate/avaliable_engines`: 获取所有可用的引擎及名称 
    - 返回参数说明
        - [json格式](#支持的引擎)
